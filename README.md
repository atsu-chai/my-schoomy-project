# 温度・土壌水分チェックプロジェクト

## これは何？
スクーミーで作った、温度と土壌水分を測定するプロジェクトです。  
測定した値に応じてRGB LEDの色を変え、シリアルモニタに温度と土壌水分の値を表示します。

## 動作内容
- 温度センサーで現在の温度を測っています。
- A1ピンにつないだ土壌水分センサーの値を測っています。
- 温度が0℃より高いと、RGB LEDが赤色に光ります。
- 土壌水分センサーの値が0より大きいと、RGB LEDが黄色に光ります。
- シリアルモニタには「温度」と「土壌水分」の値が表示されます。

## 使い方
スクーミーのプロジェクトを開き、温度センサー、土壌水分センサー、RGB LEDを接続して実行します。  
実行すると、センサーの値に応じてLEDの色が変わり、シリアルモニタで測定値を確認できます。
